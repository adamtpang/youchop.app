# Chrome Web Store Listing - Chaptr

## Extension Name
Chaptr - AI YouTube Chapters

## Short Description (132 characters max)
Instantly add AI-generated chapters to any YouTube video. Navigate videos faster with smart timestamps and descriptions.

## Detailed Description

Transform how you watch YouTube videos with Chaptr - the AI-powered chapter generator that adds smart navigation to any video in seconds.

**What Chaptr Does:**
• Automatically generates chapters with timestamps for YouTube videos
• Creates descriptive titles and summaries for each section
• Lets you jump to any part of the video instantly
• Works on any YouTube video page

**How It Works:**
1. Install the extension
2. Go to any YouTube video
3. Click the "⚡ Chapterize" button
4. Get instant chapters with one click!

**Key Features:**
✓ AI-powered chapter generation
✓ Smart timestamp detection
✓ Descriptive chapter titles
✓ Click to jump to any section
✓ Clean, intuitive interface
✓ No signup required for demo

**Perfect For:**
- Students watching educational content
- Professionals reviewing long presentations
- Anyone who wants to navigate videos efficiently
- Content creators analyzing competitor videos

**Demo Mode:**
This version includes 10 free demo credits so you can try the extension immediately. Full version with backend integration coming soon!

**Privacy:**
- No data collection in demo mode
- Local processing only
- Works entirely in your browser

**Support:**
Visit chaptr.app for documentation and support.
